About NeuroSense AI
A digital neurological screening tool aligned with Smart India Hackathon 2025 problem statement SIH25218
📋

Symptom Questionnaire

Structured self-reported assessment covering tremors, balance, memory, speech, coordination, and muscle function across six dimensions.

🧩

Memory Assessment

Object recall test measuring short-term visual memory with a controlled delay interval to evaluate retention accuracy.

⚡

Reaction Time Test

Stimulus-response measurement that evaluates neural processing speed and psychomotor reaction across multiple trials.

👆

Finger Tapping Test

Validated motor assessment measuring manual dexterity and fine motor control through rhythmic tapping velocity analysis.

📊

AI Risk Scoring

Composite weighted scoring algorithm combining all test results to generate a comprehensive neurological risk profile.

📄

Detailed Report

Downloadable PDF report containing full patient data, test outcomes, risk scores, and clinical referral recommendations.
