# NeuroSense AI

An AI-powered Early Neurological Screening Tool developed for early detection of neurological conditions.

## Features

- Symptom Assessment
- Memory Test
- Reaction Time Test
- Finger Tapping Test
- AI-Based Risk Analysis
- ESP32 + MPU6050 Sensor Integration

## Technologies Used

- HTML
- CSS
- JavaScript
- ESP32
- MPU6050
- Vercel
